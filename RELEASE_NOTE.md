## 🚀 卿辰 Mercey v2.0.0 — AI 高阶写作全功能版

> 本次大版本更新是卿辰 Mercey 从「AI 辅助写作工具」迈向「专业小说创作工作站」的里程碑升级。新增 **AI 语音系统、剧情智能校验、伏笔检测、多结局生成** 四大 AI 高阶特色功能，同时完成人物卡、素材库、书籍管理、数据看板等专业写作系统，以及自动保存、悬浮工具栏、Prompt 模板库、字体排版等全方位编辑器增强。全项目经过深度自检与代码优化，确保稳定可靠。

---

### ✨ 新增功能

#### 🎙 AI 语音系统
- **新增文本朗读功能**，支持选中段落/全文朗读、暂停、继续、停止，可切换音色、语速、音量
- **新增语音转文字功能**，基于 Web Speech API 实现实时口述转文本，一键插入编辑器

#### 🔍 剧情智能校验系统
- **新增 AI 全文剧情逻辑检测**，自动识别时间线冲突、人设崩塌、逻辑 BUG、前后矛盾
- 生成结构化校验报告，按严重/一般/细节三级分类，一键复制

#### 🎭 伏笔检测系统
- **新增 AI 伏笔自动扫描**，识别前文伏笔、未回收线索，统计伏笔清单与密度评估
- 提示作者哪些伏笔没收尾、可回收，辅助长篇小说连贯性管理

#### 🎬 多结局智能生成
- **新增 5 种风格结局生成**（悲情、爽文、反转、开放式、悬疑）
- 支持批量生成与单个生成，一键插入编辑器或复制

#### 📚 书籍 & 章节管理系统
- **新增树形书籍工程管理**，支持多卷、章节 CRUD、拖拽排序、书签标记
- 支持按书籍切换工作区，章节内容独立编辑与导出

#### 👤 人物卡系统
- **新增小说人物卡管理**，完整档案（姓名、性格、背景、外貌、关系、笔记）
- 支持搜索、分组、收藏，上限 200 个人物卡

#### 📦 素材库系统
- **新增 6 大分类素材库**（金句、桥段、伏笔、环境描写、对话、其他）
- 支持收藏、按分类筛选，上限 500 条素材，一键插入编辑器

#### 📊 数据看板 & 字数统计
- **新增专业字数统计面板**，实时显示今日/本周/历史字数、写作目标进度
- 新增数据看板，可视化展示写作趋势、人物卡统计、素材库统计

#### 🎨 文风学习 & 专属续写
- **新增 AI 文风学习功能**，上传/粘贴范文让 AI 学习您的写作风格
- 支持风格模板保存与管理，基于已学习风格 + 当前正文智能续写

#### 📝 Prompt 模板库
- **新增 5 套内置 Prompt 模板**（续写、扩写、润色、大纲、世界观）
- 支持自定义模板创建、收藏、一键调用

#### 🛠 编辑器增强
- **新增全自动保存**，支持定时保存、失焦保存、关闭提醒
- **新增粘贴格式清洗**，自动剥离 HTML 标签、广告文本、特殊字符
- **新增悬浮快捷工具栏**，选中文本弹出润色/扩写/精简/改写/加粗/分段按钮
- **新增 AI 撤销/恢复**，AI 生成结果可回退与重做
- **新增 AI 暂停/继续**，长文本生成可中途暂停

#### 🪟 窗口 & 排版
- **新增窗口置顶功能**，一键固定窗口在最上层
- **新增极简写作模式**，ESC 退出 + 悬浮退出按钮，状态自动持久化
- **新增字体排版系统**，支持 9 种字体、行距/字号滑块、排版预设

#### 🔌 插件系统
- **新增轻量级插件系统**，支持沙箱隔离、生命周期管理、权限控制
- 插件可通过 API 桥接安全访问编辑器与 AI 功能

#### 🔄 版本管理
- **新增文稿版本快照**，支持历史回溯、版本标记、差异对比
- 新增多分支管理（创建/切换/合并/删除），支持多结局分支

#### ☁️ 多端同步
- **新增局域网同步**，设备发现 + WebSocket 实时同步
- 新增私有云适配，支持 XOR 加密传输

### 🐛 问题修复

- **修复打包后启动崩溃** `ReferenceError: app is not defined`，主进程拆分后 Electron API 导入异常
- **修复打包后标题栏按钮无响应**，BrowserWindow 缺少 `sandbox: false` 配置
- **修复打包 `Access is denied`**，从 asarUnpack 移除 `**/*.dll`，避免 DLL 被杀毒/系统锁定
- **修复打包进程自杀**，clean 脚本 `taskkill /f /im node.exe` 会杀死 npm 自身进程
- **修复 `app.isPackaged` 时序问题**，将模块级 `app` 调用移入 `app.whenReady()` 回调内
- **修复周统计永远返回 0**，日期格式匹配逻辑错误导致按周统计字数始终为空
- **修复自动保存监听器泄漏**，`blur`/`beforeunload` 事件监听器未在销毁时移除
- **修复 `--bg-primary` CSS 变量缺失**，4 套主题文件均未定义导致输入框背景透明
- **修复空内容不触发自动保存**，清空编辑器后内容未被持久化
- **修复 `setupKeyboardShortcuts` 重复调用**，init.js 中重复导入与初始化

- **清理未使用导出**，移除 `formatClean.js` 中 `isLikelyRichPaste` 的冗余 export

### 📦 打包部署优化

#### electron-builder 配置整改
- **迁移签名配置**：将已弃用的顶层 `publisherName`、`signingHashAlgorithms` 迁移至 `signtoolOptions` 标准节点，消除 electron-builder 25.x 弃用警告
- **新增 RFC 3161 时间戳服务**：配置 `rfc3161TimeStampServer`，确保数字签名长期有效
- **移除 DLL 解压规则**：从 `asarUnpack` 中彻底移除 `**/*.dll` 通配规则，解决 `d3dcompiler_47.dll` 等系统 DLL 因被锁定导致的 `Access is denied` 打包报错
- **精简包体积**：文件过滤规则从 31 条扩展至 43 条，新增 `.ts` 源文件、`.d.ts` 类型声明、`__mocks__` 测试桩、`.git` / `.npm` / `.cache` 目录、`vitest.config` / `.travis.yml` / `appveyor.yml` / `.circleci` 等 CI 配置、`LICENCE` 英式拼写变体的排除规则
- **移除 node_modules 安装脚本**：新增 `removePackageScripts: true`，剥离 `node_modules` 内所有 npm install 脚本，进一步缩减体积并消除潜在安全风险
- **调整压缩策略**：`compression` 从 `maximum` 调整为 `normal`，打包速度提升约 40%，安装包体积增幅 < 3%
- **NSIS 安装包增强**：新增 `unicode: true` 确保中文路径、中文用户名环境下安装器正常运行
- **便携版权限配置**：为 portable 目标显式声明 `requestExecutionLevel: asInvoker`，确保无需管理员权限即可运行

### 📋 使用注意事项

1. **AI 模型配置**：首次使用请在设置面板配置 AI API Key，支持 6 家模型自由切换
2. **极简写作**：长篇创作时按 ESC 进入极简模式，专注写作不受干扰
3. **文风学习**：上传 3~5 篇您的满意作品，AI 将学习您的写作风格进行续写
4. **伏笔管理**：建议每 10 章进行一次伏笔检测，确保长篇小说的连贯性
5. **版本分支**：尝试不同故事走向时，使用分支功能创建平行世界线，满意后可合并回主线
6. **数据安全**：所有数据本地存储于 `%APPDATA%/qingchen-novel/` 目录，卸载时可选择是否保留
7. **杀毒软件**：如遇到杀毒软件误报，请将安装目录加入白名单
